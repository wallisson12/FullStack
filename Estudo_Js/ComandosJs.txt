    *Jquery*

    -->Para acessar um elemento do html, podemos usar

        let a = jquery("Tipo"); or let a = $("Tipo")

    --> Pega o texto no elemento, tag

        let a = $("Tipo").text();

    --> Se passar um valor nesse text, ele vai trocar o conteuno no que tem no html
        
        let a = $('Tipo').text("nome");

    --> Trabalhando com eventos, on()

        $('button').on("TipoEvento",funcao); ex(input,click, etc)

    --> O comando val(), pega o valor de um input, como o text() faz, e tambem pode mudar o valor

        let a = $('tipo').val();

    --> O comando setinterval(), faz voce fazer algom em tanto tempo
    
    --> O comando attr(), pode pegar ou adcionar elementos a minha tag

    --> O comando removeAttr(), faz a remocao do elemento da tag do html

    --> O comando one(), faz a mesma coisa que o on(), so que apenas uma vez

    --> O comando $(document).ready(function(){}), é chamada quando a pagina ja esta pronta
        obs: Uma outra maneira de usar, $(function(){logica}), vai possuir o mesmo resultado

    --> O comando $('elemento').css("elemento css","valor desse elemento"), esse vai no css desse elemento
        e vai colocar esse estilo na tag, so passando o primeiro parametro ele retorna o css 

        obs: Posso fazer de outra maneira que é, $('elemento').addclass('NomeDaClasseNoCss')
            vai adcionar essa classe de estilo nessa tag

    --> O comando removeClass('NomeDaClasseNoCss'), remove a classe da tag

    --> Com o comando $('<tr>'), podemos criar elementos de html com js

    --> Com o comando $(this).parent().parent().remove(), posso subir na arvore de tags e remover o pai  
        obs: esse this é o objeto que foi clicado e estou subindo ate o pai para retirar a linha
    
    --> O comando append() e prepend(), sempre parar inserir elementos html dentro de outros elementos
