**Comandos**

    --> Para chamar o arquivo js no html, colocar no final da tag body

        <script src="Nome Do Arquivo"></script>

    -->Executa um alert na pagina
        
        alert('Teste');
    
    --> 'document', Referencia todo o documento html, e busca o seletor desejado
        
        document.querySelector('Selector');

    --> Vai buscar o seletor, achando vai executar a funcao play  

        document.querySelector('Tag de audio').play();

    --> Sintaxe de funcao em js

        function NomeDaFuncao()
        {

        }

        obs: Pode ser adciona em uma variavel

        let teste = funcion NomeDaFuncao(){};

    --> Funcao anonima, muito utilizada quando voce precisa passar para uma variavel a funcao com um
        parametro, sem executar a funcao

        funcion(){};


    --> Seleciona uma lista de seletores e retorna uma lista

        document.querySelectorAll('selector');

    --> Acessa a classe do seletor, como se fosse um arry

        document.querySelectorAll('selector').classList[Indice]
